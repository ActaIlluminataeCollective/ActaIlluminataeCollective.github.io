We are **Arin**, **Noah**, **Hershil**, and **Andrei**—friends who wanted one calm place to host writing we stand behind: preprints, course projects, lab notes, and anything in between.

This site is a small static blog. There is no database. Each “paper” is a Markdown file in the repository, versioned like any other file on GitHub.

If you are one of us and want to publish something new, add a file under `src/posts/` with a short front matter block (title, date, author, optional abstract) and push. GitHub Actions rebuilds the site automatically.
